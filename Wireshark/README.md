# Wireshark DoS Attack Analysis Report

##  Overview

This report analyzes a Wireshark packet capture taken during a simulated Denial-of-Service (DoS) attack in a GNS3 virtual environment.

**Key Findings:**

* Total Packets: **~90,101**
* Duration: **~521.7 seconds** (~8.7 minutes)
* Primary Victim: **192.168.10.30**
* Attacker Host: **192.168.20.10**
* Attack Types Identified:

  * **TCP SYN Flood**
  * **UDP Flood**
* Additional Observations: ICMP tests, DNS failures, and normal LAN background traffic.

##  Identified DoS Attack Activity

Two distinct attack phases were detected.

#  TCP SYN Flood

### **Time Window:** ~243.1s to ~290.3s

### **Attack Rate:** ~992 packets/second

### **Protocol:** TCP

### **Target:** `192.168.10.30` (port **0**)

### **Source:** `192.168.20.10`

####  Characteristics

* Continuous **TCP SYN** packets
* Always sent to **port 0** (invalid port â€” indicates crafted attack traffic)
* Victim replies with **RST, ACK** for each packet
* No valid three-way handshake

####  Example Packets

